# Tela de Login Avançada

## Descrição

Este projeto implementa uma tela de login responsiva e moderna, desenvolvida com HTML, CSS e JavaScript, utilizando o framework Bootstrap para um design elegante e adaptável. A interface oferece uma experiência de usuário aprimorada com validações em tempo real, animações suaves e funcionalidades interativas.

## Características Principais

- Design responsivo e atraente
- Validação de formulário em tempo real
- Alternância de visibilidade da senha
- Animações CSS para feedback visual
- Simulação de processo de login

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Vite (para desenvolvimento)

## Estrutura do Projeto

- `index.html`: Estrutura da página de login
- `style.css`: Estilos e animações
- `main.js`: Lógica de interação e validação
- `package.json`: Configurações e dependências
